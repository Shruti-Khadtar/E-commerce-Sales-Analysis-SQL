# E-commerce Sales Analysis using SQL

## Project Overview
This project performs an in-depth analysis of e-commerce sales data using SQL. The goal is to extract meaningful business insights related to sales performance, customers, products, and revenue trends.

## Objectives
- Analyze total sales and revenue
- Identify top-selling products and categories
- Study customer purchase behavior
- Analyze monthly and yearly sales trends
- Practice advanced SQL queries for data analysis

## Tools & Technologies
- SQL (MySQL / PostgreSQL / SQL Server)
- CSV Dataset (E-commerce Sales Data)

## Project Structure
- **SQL** → SQL queries for analysis  
- **Dataset** → Raw e-commerce sales data  
- **Results** → Insights derived from queries  

## SQL Concepts Used
- SELECT, WHERE, GROUP BY, ORDER BY
- Aggregate Functions (SUM, COUNT, AVG)
- CASE Statements
- Subqueries
- Date-based analysis
- Joins (if applicable)

## How to Run the Project
1. Create a database in your SQL environment
2. Import the dataset into a table
3. Open `E-commerceSalesAnalysis.sql`
4. Execute queries step by step to view results

## Key Insights
- Identified top revenue-generating products
- Found peak sales months
- Analyzed repeat customer behavior
- Category-wise sales contribution
